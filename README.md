# E-Commerce
Documentation and user stories for an e-commerce web application
فكرة المشروع :
يهدف هذا المشروع إلى تصميم وتوثيق منصة تجارة إلكترونية (E-Commerce Web Application) تتيح للمستخدمين تصفح المنتجات، إدارة حساباتهم، إضافة المنتجات إلى سلة التسوق، وإتمام عمليات الشراء باستخدام طرق دفع متعددة.

كما يوفر النظام لوحة تحكم (Back-Office) للمسؤولين لإدارة المنتجات، الطلبات، المستخدمين، والعروض الترويجية، بالإضافة إلى أدوات تحليلية للمسوقين لمتابعة أداء المبيعات واتخاذ قرارات مبنية على البيانات.

يركز هذا المستودع حاليًا على مرحلة تحليل المتطلبات (Requirements Analysis) باستخدام User Stories، كنقطة انطلاق للتطوير المستقبلي
Methodology

تم اعتماد منهجية Scrum كإطار عمل لإدارة وتطوير هذا المشروع.

تعتمد Scrum على:

تقسيم المشروع إلى Sprints قصيرة (عادة من 2 إلى 4 أسابيع)

تنظيم المتطلبات على شكل Product Backlog

تمثيل المتطلبات باستخدام User Stories

التسليم التدريجي للميزات

التحسين المستمر من خلال الاجتماعات الدورية

في هذا المشروع، تم استخدام User Stories لتمثيل متطلبات النظام لكل نوع من المستخدمين (Customer, Admin, Marketer)، مما يجعلها مناسبة جدًا للعمل وفق Scrum.


سبب اختيار هذه المنهجية 

تم اختيار منهجية Scrum للأسباب التالية:

تتوافق بشكل مباشر مع User Stories المستخدمة في المشروع

مناسبة لتطبيقات التجارة الإلكترونية التي تتطلب تحديثات وتغييرات مستمرة

تساعد على تنظيم العمل وتقسيمه إلى مهام صغيرة وقابلة للتنفيذ

تشجع على التعاون بين أعضاء الفريق

تسمح بتتبع التقدم من خلال Sprints واضحة

تُستخدم على نطاق واسع في المشاريع العملية والأكاديمية

كما أن Scrum تساعد على:

تحسين جودة المنتج

تقليل المخاطر في المراحل المبكرة

تسهيل تحويل المتطلبات لاحقًا إلى مهام تطوير فعلية

التصفح والاكتشاف

As a Shopper, I want to view all products on the homepage.

As a Shopper, I want to browse products by category.

As a Shopper, I want to search for specific products.

As a Shopper, I want to filter and sort products.

As a Shopper, I want to view detailed product information.

إدارة الحساب

As a Customer, I want to create an account.

As a Customer, I want to log in and reset my password.

سلة التسوق والدفع

As a Shopper, I want to add and manage items in my cart.

As a Customer, I want to choose shipping and payment methods.

As a Customer, I want to complete checkout securely.

ما بعد الشراء

As a Customer, I want to track my orders.

As a Customer, I want to receive email confirmations.

As a Customer, I want to submit product reviews.

 Admin / Back-Office

As an Admin, I want to manage products and categories.

As an Admin, I want to manage inventory and stock.

As an Admin, I want to view and update orders.

As an Admin, I want to manage users and reviews.

As an Admin, I want to create discounts and promotions.

Marketer / Content Manager

As a Marketer, I want to publish marketing content.

As a Marketer, I want to analyze sales and product performance.
